Goals

The goals of this project can be summarised quite easily: to provide an analysis framework that is flexible, easy to use and provides the necessary performance for a short feedback cycle. In general such a framework should provide data access and restructuring (i.e Analysis Data Format -> N-tuple), basic analysis objects (PXL, HEP libraries, etc), data storage, post-processing of results (plots, tables, etc), monitoring, graphical user interface. The purpose is not to re-invent the wheel but rather package existing software and fill in the gaps.

Flexibility

In the light of developments in the Big Data analysis techniques it is crucial for a framework to be adaptive. This includes but is not limited to data formats (both input and output), analysis techniques, job submission.
